# สรุปทดสอบโปรเจค FlowAccount

## ✅ ผลทดสอบจริง (รันแล้ว)

ทดสอบด้วยเบราว์เซอร์ที่ `http://localhost:8085` (รัน `npx expo start --web --port 8085`)

| รายการ | ผล | หมายเหตุ |
|--------|-----|----------|
| หน้าแรก (ภาพรวม) | ✅ | แสดงลิงก์ "สแกนบันทึกค่าใช้จ่าย" และ "เชื่อมต่อนักบัญชี" |
| หน้าสแกนค่าใช้จ่าย | ✅ | เปิดจากปุ่มได้, มีช่องจำนวนเงิน + หมายเหตุ, กรอก 500 + ทดสอบ ได้ |
| หน้าเอกสาร | ✅ | แสดง 4 การ์ด (ใบเสนอราคา, ใบแจ้งหนี้, ใบเสร็จ, ใบกำกับภาษี) และรายการ Q-001, INV-002, REC-001, TAX-001 |
| แบบฟอร์มเอกสาร (ดูใบเสนอราคา) | ✅ | กดที่ Q-001 → เปิด `/document/view/1` แสดงหัว "แบบฟอร์มเอกสาร" |
| หน้ารายงานธุรกิจ | ✅ | เปิด `/report` ได้ แสดงหัว "รายงานธุรกิจ" |
| หน้าช่วยเหลือ | ✅ | เปิด `/help` ได้ แสดงหัว "ช่วยเหลือ" |
| หน้าสร้างใบเสนอราคา | ✅ | เปิด `/document/quote` ได้ |

**สรุป:** หน้าหลักและเส้นทาง (navigation) ทำงานครบ ฟอร์มกรอกและแบบฟอร์มดูเอกสารเปิดได้ตามที่ออกแบบ

## วิธีรันแอป

```bash
cd c:\Users\USER\FlowAccount
npm install   # ครั้งแรกเท่านั้น
npx expo start
```

## สิ่งที่โปรเจคทำได้ (รายการทดสอบ)

### 1. ภาพรวม (Dashboard)
- [ ] แสดงยอดคงเหลือเดือนนี้ (คำนวณจากรายรับ − รายจ่าย)
- [ ] แสดงการ์ดรวมรายรับ และรวมรายจ่าย
- [ ] ปุ่ม **สแกนบันทึกค่าใช้จ่าย** → เปิดหน้าสแกนค่าใช้จ่าย
- [ ] ปุ่ม **เชื่อมต่อนักบัญชี** → เปิดหน้าคำนวณ/นักบัญชี
- [ ] แสดง **รายการล่าสุด** 5 รายการ (จากข้อมูล mock + ที่เพิ่มจากสแกน)

**วิธีทดสอบ:** เปิดแอป → ดูตัวเลขบนการ์ด → กดปุ่มทั้งสอง → ดูรายการล่าสุด

---

### 2. รายรับ-รายจ่าย
- [ ] แท็บ **ทั้งหมด / รายรับ / รายจ่าย** สลับได้
- [ ] แสดง **รวมรายรับ** และ **รวมรายจ่าย**
- [ ] ช่อง **ค้นหารายการ** กรอกแล้วกรองรายการได้
- [ ] รายการแสดงจากข้อมูลในแอป (mock + ที่เพิ่มจากสแกน)

**วิธีทดสอบ:** ไปแท็บ "รายรับ-รายจ่าย" → สลับแท็บ → พิมพ์คำค้นในช่องค้นหา

---

### 3. สแกนค่าใช้จ่าย
- [ ] ปุ่ม **ถ่ายรูป** / **เลือกจากคลัง** เลือกรูปใบเสร็จได้
- [ ] กรอก **จำนวนเงิน** และ **หมายเหตุ**
- [ ] กด **บันทึกค่าใช้จ่าย** → แสดง Alert สำเร็จ แล้วกลับ
- [ ] รายการใหม่ไปโผล่ใน **ภาพรวม** และ **รายรับ-รายจ่าย**

**วิธีทดสอบ:** จากภาพรวมกด "สแกนบันทึกค่าใช้จ่าย" → กรอกเงิน (เช่น 500) + หมายเหตุ → บันทึก → กลับไปดูภาพรวม/รายรับ-รายจ่าย

---

### 4. เอกสาร (ใบเสนอราคา, ใบแจ้งหนี้, ใบเสร็จรับเงิน, ใบกำกับภาษี)
- [ ] การ์ด 4 ประเภท: ใบเสนอราคา, ใบแจ้งหนี้, ใบเสร็จรับเงิน, ใบกำกับภาษี
- [ ] กดการ์ด → เปิด **ฟอร์มสร้างเอกสาร** (ลูกค้า, รายการ, จำนวนเงิน)
- [ ] เลขที่เอกสารสร้างอัตโนมัติ (Q-002, INV-003, REC-002, TAX-002 ฯลฯ)
- [ ] กดบันทึก → เอกสารเพิ่มในรายการ "เอกสารล่าสุด"
- [ ] กด **รายการในเอกสารล่าสุด** → เปิด **แบบฟอร์มเอกสาร** (ใบเสนอราคา/ใบอื่น) แสดงหัวเอกสาร, ลูกค้า, ตารางรายการ, รวมเงิน, เงื่อนไข (ใบเสนอราคา), ลายเซ็น

**วิธีทดสอบ:** ไปแท็บ "เอกสาร" → กด "ใบเสนอราคา" → กรอกลูกค้า + รายการ + จำนวนเงิน → บันทึก → กลับมาที่เอกสาร → กดรายการที่เพิ่งสร้าง → ดูแบบฟอร์มใบเสนอราคา

---

### 5. รายงานธุรกิจ
- [ ] **แดชบอร์ดการเงิน**: ยอดคงเหลือ, รวมรายรับ, รวมรายจ่าย
- [ ] **สรุปยอดรายเดือน** (mock 3 เดือน)
- [ ] **รายงานสรุปยอดขายและภาษี**: ยอดขายจากเอกสาร, ภาษี 7%, รวม
- [ ] **สรุปเอกสาร** แยกตามประเภท (ใบเสนอราคา, ใบแจ้งหนี้, ใบเสร็จ, ใบกำกับภาษี)

**วิธีทดสอบ:** ไป "เพิ่มเติม" → "รายงานธุรกิจ" → เลื่อนดูแต่ละบล็อก

---

### 6. เชื่อมต่อนักบัญชี
- [ ] อธิบายวิธีเชื่อมต่อ 3 ขั้นตอน
- [ ] กรอกอีเมล → กด **เชิญนักบัญชี** → แสดง Alert (Demo) และเพิ่มชื่อในรายการ
- [ ] **หาสำนักงานบัญชี**: บล็อกข้อความ + ปุ่ม "ค้นหาสำนักงานบัญชี" (Demo)
- [ ] **นักบัญชีที่เชื่อมต่อแล้ว**: แสดงรายชื่อที่เชิญ (mock)

**วิธีทดสอบ:** ไป "เพิ่มเติม" → "เชื่อมต่อนักบัญชี" → กรอกอีเมล → กดเชิญ → ดูรายการด้านล่าง

---

### 7. ตั้งค่า
- [ ] แสดงเมนูตั้งค่า (ข้อมูลธุรกิจ, การแจ้งเตือน, ภาษา, โหมดมืด) แบบ Demo

**วิธีทดสอบ:** ไป "เพิ่มเติม" → "ตั้งค่า"

- [ ] **ติดต่อเรา**: 02-026-8989, support@flowaccount.com, 02-026-8991 / demo@flowaccount.com

**วิธีทดสอบ:** ไป "เพิ่มเติม" → "ช่วยเหลือ" → อ่าน FAQ และบล็อกติดต่อ
