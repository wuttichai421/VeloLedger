# ลิงก์ทุกหน้าของแอป FlowAccount

ใช้กับ base URL: `http://localhost:8084` (หรือโดเมนจริงเมื่อ deploy)

## แท็บหลัก (Tab Bar)
| เมนู | URL |
|------|-----|
| ภาพรวม | `/` หรือ `/(tabs)` |
| รายรับ-รายจ่าย | `/income-expense` |
| เอกสาร | `/documents` |
| เชื่อมต่อ | `/connect` |
| เพิ่มเติม (เมนู Navbar) | `/more` – แค่รายการเมนู ไม่ใช่หน้าเต็ม |

## จากเมนู "เพิ่มเติม" (มี Navbar ตลอด)
| เมนู | URL |
|------|-----|
| สแกนบันทึกค่าใช้จ่าย | `/scan-expense` |
| เอกสาร | `/documents` |
| เชื่อมต่อนักบัญชี | `/connect` |
| รายงานธุรกิจ | `/more/report` |
| แพ็กเกจและราคา | `/more/pricing` |
| ตั้งค่า | `/more/settings` |
| ช่วยเหลือ | `/more/help` |
| การแจ้งเตือนเอกสาร | `/more/reminders` |
| โหมดมืด (ตั้งค่า) | `/more/settings` |
| บทความบัญชี ภาษี | `/more/articles` |
| วิดีโอสอนใช้งาน | `/more/videos` |

## หน้ารายการเอกสาร (จากหน้า เอกสาร)
| ประเภท | URL |
|--------|-----|
| ใบเสนอราคา | `/document/quote` |
| ใบแจ้งหนี้ | `/document/invoice` |
| ใบเสร็จรับเงิน | `/document/receipt` |
| ใบกำกับภาษี | `/document/tax_invoice` |
| ใบวางบิล | `/document/delivery_order` |

## หน้ามุมมองเอกสาร (จากรายการหรือการแจ้งเตือน)
| รูปแบบ | URL |
|--------|-----|
| ดู/แก้ไขเอกสารตาม id | `/document/view/[id]` เช่น `/document/view/1` |

---

**หมายเหตุ:** ทดสอบการเปิดทุก URL แล้ว (เมื่อรัน `npm run web` ที่พอร์ต 8084) ครบทุกหน้า
